# crayon-micro-apps

一个基于 Single-Spa 的 VUE 微前端示例

## 开始

```
cd crayon-micro-root
npm install
npm run serve
```

### NavBar

```
cd crayon-micro-navbar
npm install
npm run serve
```

### Account

```
cd crayon-micro-account
npm install
npm run serve
```

### Products

```
cd crayon-micro-products
npm install
npm run serve
```

在浏览器里面输入：`http://localhost:9010/`
